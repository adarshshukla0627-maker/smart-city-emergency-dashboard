# Smart City Emergency Dashboard

This dashboard provides real-time monitoring and management for city emergencies, including incident tracking, ambulance dispatch, analytics, AI assistance, and interactive maps.

Original Figma: https://www.figma.com/design/eW8SxK5LXHeyLvpDKbAVfp/Smart-City-Emergency-Dashboard

## Features
- Interactive Leaflet maps with emergency markers
- Real-time alerts and incidents panel
- Analytics cards and charts (Recharts)
- AI terminal/command interface
- Ambulance tracking
- Admin panel
- Responsive UI with shadcn/ui + Tailwind + Radix
- Dark/light theme support

## Tech Stack
- React 18
- Vite
- React Router
- Tailwind CSS
- React Leaflet
- Recharts
- React Hook Form
- Zod (inferred)
- Lucide icons
- Sonner toasts

## Quick Start
1. `npm install`
2. `npm run dev`
3. Open http://localhost:5173

## Scripts
- `npm run dev` - Start dev server
- `npm run build` - Build for production
- `npm run preview` - Preview production build (if configured)

## Pages/Routes
- `/` - HomePage
